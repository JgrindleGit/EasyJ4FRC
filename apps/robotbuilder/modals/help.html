What can we help you with?

  • Where should I begin?
    Try loading one of the examples.
    From the File menu choose Examples.. Try the Simple Robot Example
  • A Specific Block
    The best way to learn more about what a block does it to hover your mouse pointer over it, for most blocks one will see a small bit of hovertext detailing the usage of the block.
  • Deploying Code
    1. @todo Revise this
    2. Navigate to the script tab. (Just left of help tab)
    3. Copy the script the program has generated for you
    4. Open Netbeans on your computer. (Don't have it? View our Getting Started Guide)
    5. File > New Project
    6. Select FRC Java (Is it Missing? You might have missed step 5 in our Getting Started Guide)
    7. In the right pane, select SimpleRobotTemplateProject
    8. Then click the Next > Button
    9. Name your project (Best to avoid spaces, punctuation and symbols. Stick to numbers and letters)
    10. Pick a location to store your new project (ie: My Documents)
      11. ***It is important to enter the following EXACTLY as they appear. (Just the Italicized part)
    11. Project Package: edu.team2648.frcEasyJ
    12. Robot Class: MyRobot
    13. Click the Finish button. Netbeans will now create your robot project.
    14. On the left side of your screen you should see the project you just created with two things under it. src and build.xml
    15. Click the small plus next to src
    16. Then the plus next to edu.team2648.frcEasyJ
    17. Now you should see MyRobot.java : Double click it (Open for editing)
    18. This is the default code that is provided for you for simple robot projects. We are going to replace it with the code generated for us.
    19. Edit > Select All
    20. Edit > Paste
    21. Now we should save our new project. File > Save
    22. You are now ready to download code to your robot.

    Check out our Getting Started Guide to learn to download code to your cRio robot controller.
  • Copying Blocks
    One can easily copy blocks by right clicking the block they wish to copy, and choosing duplicate.
    NOTE: all blocks that are contained within the block will also be copied.
